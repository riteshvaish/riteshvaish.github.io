Summary : Latitude Vs . X


The Purpose of this project was to analyze how the weather changes as the latitude of a place changes and we go closer to equator. So for this analysis we first pulled the data from OpenWeatherMap API to assemble a dataset on over 500 cities with their coordinates.

After Assembling the data we used Matplotlib to pilot various aspectss of the weather vs Latitude. Here during our analysis the weather factors we looked at, included temperature, Cloudiness, wind speed and humidity. This site provided the source data and visualizations created as a part of analysis, as well as explainations and descriptions of any trends and correlations witnessed.
